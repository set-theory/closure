# Installation

Can be managed using
[yarn](https://yarnpkg.com/en/docs),
[npm](https://docs.npmjs.com),
or [jspm](https://jspm.org/docs).


### yarn
```terminal
yarn add @set-theory/closure
```

### npm
```terminal
npm install @set-theory/closure --save
```

### jspm
```terminal
jspm install npm:@set-theory/closure
```
